How to Use

  • Scan: Tap START SCANNER and point the camera at the barcode. Closes automatically after a successful read — device vibrates and beeps to confirm.
  • Manual input (17 digits): Type or paste the 17-digit SSCC body — the check digit is calculated and appended automatically when you press AUDIT.
  • Manual input (18 digits): Type or paste a full 18-digit SSCC to validate it. A wrong check digit is highlighted in red and the correct value is shown.
  • Multiple codes: Enter one code per line. Press AUDIT to validate all at once.
  • Range: Enter two SSCCs separated by - to generate every code in between with check digits, e.g. 34001234500000001-34001234500000050. Supports 17, 18 and 20-digit formats with or without AI (00) prefix. Max 500 codes.
  • File upload: Click the upload area or drag and drop any EDIFACT, IDoc SAP, XML, CSV or TXT file — all SSCCs are extracted and validated automatically. 100% client-side, no data leaves your device.
  • Copy & Export: Use the copy button on each row to copy a single SSCC, or tap CSV in the summary bar to download all results.
