Scan barcodes, validate or generate SSCCs, expand ranges, or upload EDIFACT/XML/CSV files.

Show all instructions
  • Scan: Tap START SCANNER and point the camera at the barcode.
  • Manual input: Type or paste 17 digits to generate the check digit, or 18 to validate.
  • Multiple codes: Enter one code per line. Press AUDIT to process all.
  • Range: Use START-END to generate up to 500 codes.
  • File upload: Drag and drop EDIFACT, XML or CSV files for automatic extraction.
