REST API

SSCC API

Free, open REST API for SSCC validation, check digit generation, range expansion and file extraction. No authentication required.

https://sscc.birth1mark.workers.dev
Auth
None required
Rate limit
100k req/day
CORS
* (open)
Format
JSON
Max range
500 codes
Uptime
Cloudflare edge

Endpoints

GET
/validate?sscc=<18 digits>
Validate an 18-digit SSCC — checks the GS1 check digit and returns country info
GET
/generate?body=<17 digits>
Calculate and append the GS1 check digit to a 17-digit SSCC body
GET
/range?from=<SSCC>&to=<SSCC>
Generate every SSCC between two codes with check digits (max 500)
GET
/prefix?sscc=<18 digits>
Look up the GS1 member country from the SSCC company prefix
POST
/extract
Extract and validate all SSCCs from EDIFACT, IDoc SAP, XML, CSV or TXT content sent in the request body
GET
/health
API status and endpoint listing

GET /validate

Validates an 18-digit SSCC. Accepts bare 18-digit strings or 20-digit strings with the GS1 AI (00) prefix.

curl "https://sscc.birth1mark.workers.dev/validate?sscc=356012345600000016"
const res  = await fetch('https://sscc.birth1mark.workers.dev/validate?sscc=356012345600000016');
const data = await res.json();
console.log(data.valid);       // true
console.log(data.gs1.country); // "Portugal"
import requests
r = requests.get('https://sscc.birth1mark.workers.dev/validate',
                 params={'sscc': '356012345600000016'})
print(r.json()['valid'])       # True
print(r.json()['gs1']['country']) # Portugal
200 OK application/json
{
  "input":     "356012345600000016",
  "sscc":      "356012345600000016",
  "formatted": "(00)356012345600000016",
  "valid":     true,
  "checkDigit": { "provided": 6, "expected": 6 },
  "gs1": {
    "extensionDigit": 3,
    "prefix":         560,
    "country":        "Portugal",
    "countryCode":    "PT",
    "flag":           "🇵🇹"
  }
}

GET /generate

Calculates the GS1 check digit for a 17-digit SSCC body and returns the complete 18-digit SSCC.

curl "https://sscc.birth1mark.workers.dev/generate?body=35601234560000001"
const res  = await fetch('https://sscc.birth1mark.workers.dev/generate?body=35601234560000001');
const data = await res.json();
console.log(data.sscc); // "356012345600000016"
r = requests.get('https://sscc.birth1mark.workers.dev/generate',
                 params={'body': '35601234560000001'})
print(r.json()['sscc']) # 356012345600000016
200 OK application/json
{
  "input":      "35601234560000001",
  "body":       "35601234560000001",
  "checkDigit": 6,
  "sscc":       "356012345600000016",
  "formatted":  "(00)356012345600000016"
}

GET /range

Generates every SSCC between two codes, each with its check digit. Accepts 17, 18 or 20-digit inputs on either side.

curl "https://sscc.birth1mark.workers.dev/range?from=35601234560000001&to=35601234560000005"
const res  = await fetch('https://sscc.birth1mark.workers.dev/range?from=35601234560000001&to=35601234560000005');
const data = await res.json();
data.items.forEach(item => console.log(item.sscc));
r = requests.get('https://sscc.birth1mark.workers.dev/range',
                 params={'from': '35601234560000001', 'to': '35601234560000005'})
for item in r.json()['items']:
    print(item['sscc'])
200 OK application/json
{
  "from":  "35601234560000001",
  "to":    "35601234560000005",
  "count": 5,
  "items": [
    { "sscc": "356012345600000016", "formatted": "(00)356012345600000016", "checkDigit": 6 },
    { "sscc": "356012345600000023", "formatted": "(00)356012345600000023", "checkDigit": 3 },
    ...
  ]
}

POST /extract

Send the raw content of any EDIFACT, IDoc SAP, XML, CSV or TXT file as the request body. The API detects the format automatically and returns all SSCCs found, validated against the GS1 prefix table and check digit.

Format detection: EDIFACT is detected by a UNA or UNB+ header. XML/IDoc by a <?xml or <IDOC> tag. Everything else is scanned with a generic regex pipeline. 
curl -X POST "https://sscc.birth1mark.workers.dev/extract" \
  -H "Content-Type: text/plain" \
  --data-binary @desadv.edi
// From a file input
const file    = document.querySelector('#file-input').files[0];
const content = await file.text();
const res     = await fetch('https://sscc.birth1mark.workers.dev/extract', {
    method: 'POST',
    body:   content,
});
const data = await res.json();
console.log(data.count);  // number of SSCCs found
console.log(data.items);  // array of validated SSCCs
with open('desadv.edi', 'r') as f:
    content = f.read()

r = requests.post('https://sscc.birth1mark.workers.dev/extract', data=content)
data = r.json()
print(f"Found {data['count']} SSCCs ({data['format']} format)")
for item in data['items']:
    print(item['sscc'], item['valid'])
200 OK application/json
{

Errors

All errors return a JSON object with an error field and an appropriate HTTP status code.

400 Bad Request
{ "error": "Invalid length: expected 18 digits, got 15" }
